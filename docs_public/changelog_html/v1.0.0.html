üë§ Character Management NEW

/addcharacter

Create a new character with your Tibia stats. (Later renamed to /character add)

Required fields:

  • Character name
  • Level
  • Max HP
  • Max Mana
  • Select your unlocked charms

/character view

View all your saved characters and their stats.

/character edit

Update your character's stats or charms.

/character delete

Remove a character and all its saved data.

Limits: Up to 15 characters per user

üìä Hunt Analysis UPDATED

/analysis

Analyze your hunt using your character's saved charms.

Changes from v0.1:

  • Now uses charms from your saved character
  • Select a character before pasting Hunt Analyzer data
  • Can save analysis results for future reference
  • All charms use a fixed 10% proc rate
Limits: Up to 20 saved analyses per character

/bestcharms

Find the best charm configuration using all available damage charms.

Changes from v0.1:

  • Now linked to a character (uses character stats for calculations)
  • Can save results for future reference
  • All charms use a fixed 10% proc rate
Limits: Up to 20 saved bestcharms per character

üìã Hunt Management NEW

/hunt view

View your saved analyses and bestcharms configurations.

Flow:

  1. Select a character
  2. Choose hunt type (analysis or bestcharms)
  3. Select which saved hunt to view

/hunt delete

Delete a specific saved hunt.

/hunt rename

Rename a saved hunt.

/weaponxp

Calculate weapon proficiency XP earned during your hunt.

Features:

  • Paste Hunt Analyzer data

Calculate ingredients needed for the Hot Cuisine Quest.

Features:

  • Support for first-time completion (all dishes)
  • Support for repeat completions (select dishes)
  • Multiplier for multiple characters
  • Ingredients grouped by city and NPC
  • Links to TibiaWiki for NPCs and locations
